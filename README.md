# Library Management System using JDBC

## Description

A Java-based Library Management System developed using JDBC and MySQL. The system allows users to manage books, perform CRUD operations, borrow books, and return books.

## Features

- Add Books
- View Books
- Update Book Details
- Delete Books
- Borrow Books
- Return Books
- JDBC Database Connectivity
- MySQL Integration

## Technologies Used

- Java
- JDBC
- MySQL
- Object-Oriented Programming
- SQL

## How to Run

1. Create Database using Database.sql
2. Configure JDBC Connection
3. Compile:

```bash
javac *.java
```

4. Run:

```bash
java LibraryManagementSystem
```
